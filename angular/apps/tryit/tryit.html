Angular Samples

  • AngularJS was created by Google & version 1.0 was released in 2012
  • AngularJS extends JS with "ng" directives
    • The ng-app directive defines an AngularJS application.
    • The ng-model directive binds the value of HTML inputs to application data.
    • The ng-bind directive binds application data to the HTML view.
    • The ng-repeat iterates through data in a collection.
    • ng-options is used to populate a select drop down list.
    • ng-non-bindable allows the display of "{{x}}" otherwise it would be evaluated.
  • AngularJS starts automatically when the web page has loaded.
  • Include the script src angular.min.js
  • Unlike JS, angular expressions can be written directly inside HTML
  • Add controllers, directives, filters, and more
