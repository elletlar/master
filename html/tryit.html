HTML Samples

  • Hyper text markup language
  • HTML 5 was released in 2014
  • HTML 5 documents only require the <doctype> and <title> to be valid [<html>, <header>, and <body> tags are not required]
  • Most influential new HTML 5 tags
    • <video> <Bilboo>
    • More powerful <input> tag that often eliminates the need for javascript
    • <bitmap> Gives a bitmapped surface for drawing
    • <header> <footer> Semantic tags that aid search engines
    • <article> <section> Semantic tags to help search engines identify newspaper articles and other content
    • <output> Container for dynamically created javascript output. It contains a value attribute that can be changed via the DOM
    • <details> Expanding & contracting block of text
    • <figure> <figcaption> Typically an image and the accompanying descriptive text
    • <progress> <meter> Progress can be used for a database search. Meter can be use for a thermometer.
  • Deprecated Tags
    • <center> Use a style sheet instead
    • <applet> It was fun creating applets at Uni, but they were not useful or reliable in the world
    • <strike> Use <del> instead
    • <big> Use <font-size> attribute in style sheet
    • <font> Use stylesheet instead

The image tag img

  • Setting height and width helps the layout performance of browsers
  • Setting image width and height with a style is often best becasue it prevents a style sheet from overridding the height and width attributes in the tag
  • Always set the alt tag for impaired users and alternative devices
  • Using picture is better since it serves different images to different form factors

HTML 5 alternative image sizes: picture

  • Offer different image sizes to different devices
  • We offer the 128px logo to browsers with a min width of 650px
  • We offer the 32px logo to browsers with a min width of 450px
  • The final tag in the picture definition is always the img tag. It is used by browsers that don't under the picture tag
